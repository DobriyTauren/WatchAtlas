# WatchAtlas

WatchAtlas is a personal movie and TV series tracker built as a client-side web application. It focuses on clean browsing, episode progress tracking, local-first data storage, and lightweight analytics for watch history and library management.
The application runs directly in the browser and is deployed via **GitHub Pages**.

🔗 [Try the application](https://dobriytauren.github.io/WatchAtlas/)

## Features

- Track movies, series, seasons, and episodes
- Add a full TV series from Markdown, to speed up data entry and bulk library setup
- Mark individual seasons or full series as watched
- Browse the library in grid or list layouts
- View statistics for watch time, progress, and series rankings
- Export and restore local backups

## Technologies

- C# / .NET 10
- Blazor WebAssembly
- Razor Components
- Tailwind CSS
- JavaScript interop for browser storage, downloads, and theme bootstrap
- LocalStorage-based persistence
- xUnit for automated tests
